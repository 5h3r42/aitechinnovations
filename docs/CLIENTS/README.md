# Client and Prospect Records

Create one Markdown file per prospect or client, using a safe filename such as `business-name.md`. Do not create a record until there is verified information to add.

## Record structure

```md
# [Business]

## Business

[Business name]

## Contact

[Name, role, and approved contact details]

## Website

[URL]

## Industry

[Industry]

## Current Issues

[Verified observations]

## Demo Shown

[Demo or relevant example]

## Proposal

[Proposal status, link, or notes]

## Follow-up

[Next action, owner, and date]

## Outcome

[Open / won / lost / nurture / do not contact]
```
