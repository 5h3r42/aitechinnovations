# Sales Pipeline

Use the following statuses consistently in `prospect-tracker.csv`.

| Status | Meaning | Required next action |
| --- | --- | --- |
| Researching | Business is being assessed; no contact made. | Verify fit and observation. |
| Ready to contact | Research is complete and outreach is approved. | Send the selected message. |
| Contacted | Initial outreach has been sent. | Set follow-up date. |
| Engaged | Prospect has replied or started a conversation. | Qualify and agree next step. |
| Call booked | Discovery call is confirmed. | Prepare call notes. |
| Proposal sent | Tailored proposal has been shared. | Set decision follow-up. |
| Won | Prospect has confirmed and onboarding begins. | Start onboarding checklist. |
| Lost | Opportunity is closed without a sale. | Record the reason if known. |
| Nurture | Not ready now but appropriate for later contact. | Set a permission-based future follow-up. |
| Do not contact | Prospect requested no further contact or is unsuitable. | Stop outreach. |

## Data hygiene

- Update the status, next follow-up date, outcome, and notes after every meaningful interaction.
- Keep observations factual and concise.
- Do not store unnecessary sensitive personal information.
